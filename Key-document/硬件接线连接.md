1. LD2450 毫米波雷达 (Radar)
 
- 通信接口：SCI2 / UART2
- 接线定义：
- RA6M5  P301 (RXD2)  ← 雷达  TX 
- RA6M5  P302 (TXD2)  → 雷达  RX 
- 通信参数：波特率  256000 ，数据位  8 ，停止位  1 ，无校验（ 8N1 ）
 
 
 
2. OpenMV 视觉摄像头 (Vision)
 
- 通信接口：SCI9 / UART9
- 接线定义：
- RA6M5  P601 (RXD9)  ← OpenMV  TX (P4引脚) 
- RA6M5  P602 (TXD9)  → OpenMV  RX (P5引脚) 
- 通信参数：波特率  115200 ， 8N1 
 
 
 
3. ST-3215 双轴舵机 (通过 URT-1 扩展板)
 
- 通信接口：SCI5 / UART5
- 接线定义：
- RA6M5  P501 (TXD5)  → URT-1  TX    (无需怀疑，具体原因可见：algorithm-optimization\References\ST-3215-C018(12V-30K-Gear-ratio-1-34)\FE-URT-1调试板\URT-1转换电路原理图)
- RA6M5  P502 (RXD5)  ←  URT-1  RX
- 通信参数：波特率  1000000 （1M 高速串口）， 8N1 ，用于实现低延迟控制
-另：蓝色端子(DC6v-9v):G,V1端接稳压模块DC(12V)和GND端 ；三色总线(白红黑)接S，v1,G再接舵机 (两舵机之间的连接见algorithm-optimization\References\ST-3215-C018(12V-30K-Gear-ratio-1-34)\FE-URT-1调试板\连接示意图）
 
4. ESP32 异步遥测网关 (Gateway)
 
- 通信接口：SCI3 / UART3 (RA6M5) ↔ UART2 (ESP32)
- 接线定义（交叉对接）：
- RA6M5  P707 (TXD3)  → ESP32  GPIO16 (RXD2) 
- RA6M5  P706 (RXD3)  ← ESP32  GPIO17 (TXD2) 
- 通信参数：波特率  115200 ， 8N1 

 
 
 
💡 开发提示
 
- 所有外设均采用 UART 异步串行通信，需在 RA6M5 侧初始化对应 SCI 通道，并匹配好波特率与帧格式。
- 舵机使用 1M 波特率，需确保主控时钟配置能支持该速率，避免通信误码。
- ESP32 网关链路需特别注意引脚交叉，否则会导致通信单向或完全失效。
