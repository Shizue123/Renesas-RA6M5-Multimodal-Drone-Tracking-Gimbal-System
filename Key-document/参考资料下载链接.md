## 5. 稳压模块

本项目使用 **RUNESKEE 大电流多路 3.3V/5V/12V 电源模块**，参数如下：

| 参数 | 值 |
|------|-----|
| 输入电压 | 6V–12V（如需 12V 输出，供电必须 12V） |
| 输入电流 | ≤ 3A（电池供电时容量需 ≥ 3000mAh） |
| 3.3V 输出 | Imax = 1A |
| 5V 输出 | Imax = 3A |
| 12V 输出 | Imax = 3A（直通，供电 12V 则输出 12V） |
| 电源输入接口 | 接线端子 / DC5.5×2.1 |
| 附加功能 | 5V USB 输出口、电源开关、供电防反接保护（反接亮红灯警告） |

> 舵机供电：URT-1 蓝色端子 V1+G 接本模块 12V 输出和 GND。
